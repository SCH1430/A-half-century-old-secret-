# Release Candidate (RC1)

Версия: **RC1.1**  
Дата сборки: 2026-02-12

**Авторы:** Марыкин Тимофей, Максим Трушин  
**Консультант:** Руководитель музея истории создания противоракетной обороны «Звезды в пустыне» ГБОУ г. Москвы «Школа № 1430», Давидович Анна Павловна.

## Что сделано
### Дизайн
- Единый «язык панелей» (фон/обводка/тени) для квеста, оверлеев и ключевых карточек.
- Мягкая унификация кнопок (радиусы/обводка/фокус) без ломания существующего фирменного стиля.

### Квест
- Таймлайн: добавлен **scroll-snap**, увеличена удобная зона клика, добавлен лёгкий fade по краям (видно, что можно скроллить).
- Вся навигация остаётся под сценой (как часть карточки), без перекрытий.

### Полигон испытаний
- HUD стал чище: «Осталось: 0» автоматически скрывается, когда волна не активна.

### Управление (правый блок)
- Кнопки **?** и **звук** всегда квадратные и симметричные (даже при раскрытой панели звука).
- Исправлены конфликтующие мобильные правила (`width: 100%` для sound toggle).

### PWA
- Обновлена версия Service Worker: `p1430-pwa-rc1` — гарантированное обновление в установленном приложении.

## Быстрый чек
- Открытие на 480 / 390 / 360 px: без наездов и критичных переполнений.
- Запуск по HTTPS/GitHub Pages: PWA работает и обновляется.
